安装教程

六步开启智能之旅

1

下载并打开 Kiro IDE

从下载中心获取 Kiro IDE 安装包，双击安装完成后打开。首次启动会进入登录界面，点击 Sign in 按钮开始登录。

  • 支持 Windows / macOS / Linux
  • 双击安装包按提示完成安装
  • 首次打开进入登录界面
  • 点击紫色 Sign in 按钮

点击 Sign in 后，Kiro 会自动跳转到默认浏览器进行登录。如果没有自动跳转，可以手动复制链接在浏览器中打开。

  • 自动打开默认浏览器
  • 如未跳转可手动复制链接
  • 登录完成后自动返回 IDE

  • GitHub — 无需科学上网，推荐使用
  • Builder ID — AWS 开发者账号
  • Your organization — 企业 SSO 登录

5

安装代理插件

点击左侧栏的扩展图标（方块图标）打开扩展面板，通过 "..." 菜单选择 Install from VSIX，安装下载好的插件文件。

  • 点击左侧栏扩展图标
  • 点击右上角 "..." 菜单
  • 选择 Install from VSIX
  • 选择下载的 .vsix 文件完成安装

6

打开插件并激活

安装完成后，点击左侧栏的闪电图标进入插件管理面板，输入您获取的激活码，点击「激活」按钮即可享受无限 AI 能力。

  • 点击左侧栏闪电 ⚡ 图标
  • 在输入框粘贴激活码
  • 点击「激活」按钮
  • 等待自动注入完成即可使用
